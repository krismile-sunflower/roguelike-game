# 后续计划

## 立即验证

1. 用 Godot 编辑器运行 `res://scenes/main.tscn`。
2. 从主菜单开始新游戏，至少通关前两层。
3. 检查所有按钮、暂停、重开本层、失败和继续游戏流程。
4. 打开 Debugger，确认没有脚本报错和缺失资源。

## 玩法增强

1. 增加视野和战争迷雾，让探索更有 roguelike 张力。
2. 给不同敌人加入独立行为：远程攻击、绕路、逃跑、守门。
3. 加入可选择升级，例如生命、攻击、暴击、回血效率。
4. 让 boss 层有独特机制，而不仅是高生命敌人。
5. 加入更多房间模板和特殊房间：宝库、陷阱房、治疗泉、锁门房。

## 技术整理

1. 将 `scripts/main.gd` 拆分为地图生成、实体、战斗、UI 四个模块。
2. 将楼层和敌人配置移动到独立资源或 JSON。
3. 增加自动化冒烟测试脚本，验证地图总能生成可达出口。
4. 清理不再参与主流程的历史脚本和场景，或移动到 archive 目录。

## 素材与呈现

1. 为主角、敌人和出口补充简单动画。
2. 增加背景音乐和楼层环境音。
3. 制作正式标题图标、桌面图标和导出 preset。
4. 根据 Kenney Tiny Dungeon 的瓦片风格补齐 UI 框、按钮和状态图标。
